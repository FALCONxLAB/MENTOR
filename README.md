# MENTOR Project Page

This is a static GitHub Pages project page.

## File structure

```text
.
├── index.html
├── .nojekyll
├── README.md
└── static
    ├── css
    │   └── index.css
    ├── images
    │   ├── logo.svg
    │   ├── show.svg
    │   └── architecture-placeholder.svg
    └── videos
        ├── .gitkeep
        └── demo.mp4  ← put your own video here
```

## Replace these files

- Put your main video at `static/videos/demo.mp4`.
- Replace `static/images/show.svg` with your poster image if needed.
- Replace `static/images/architecture-placeholder.svg` with your architecture figure if needed, or update the image path in `index.html`.

## Publish with GitHub Pages

1. Create a repository under `FALCONxLAB`, for example `MENTOR`.
2. Upload all files in this folder to the repository.
3. Go to **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Select branch **main** and folder **/root**.
6. Visit `https://falconxlab.github.io/MENTOR/`.
